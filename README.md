# 2048 Game

一个用 JavaScript 编写的经典 2048 游戏。

## 功能特性

- 🎮 使用方向键或 WASD 移动方块
- 💯 实时显示得分和最高分
- 🎯 平滑的动画效果
- 📱 响应式设计，支持移动设备
- 💾 本地存储最高分

## 如何开始

1. 克隆或下载仓库
2. 在浏览器中打开 `index.html` 文件
3. 使用方向键或 WASD 开始游戏

## 游戏规则

- 使用方向键或 WASD 移动所有方块
- 相同数字的方块相撞时会合并成一个更大的数字
- 每次移动后会在空白处生成一个新的 2 或 4
- 当数字达到 2048 时获胜
- 如果所有格子都被填满且无法移动，游戏结束

## 技术栈

- HTML5
- CSS3 (Grid + Animations)
- 原生 JavaScript (ES6+)
